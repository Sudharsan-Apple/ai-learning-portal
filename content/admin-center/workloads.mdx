# Exchange & SharePoint Admin Essentials

## 1. Exchange Online Admin Center
Manage email objects and policies.
- **User Mailboxes:** Primary email for individuals.
- **Shared Mailboxes:** Used by groups (e.g., `info@contoso.com`).
- **Distribution Lists:** Group email address for bulk communication.
- **Security Groups:** Used for email *and* permissions (SharePoint, Teams).

## 2. SharePoint Admin Center
The "Engine Room" for data storage in Microsoft 365.
- **Sites:** Individual sites for departments, projects, or teams.
- **Libraries:** Folders for documents and metadata.
- **Permissions:** Full Control, Design, Edit, Contribute, Read, and View Only.
- **Guest Access:** External sharing controls.

## 3. Microsoft Teams Admin
- **Teams:** Created for collaboration.
- **Channels:** Standard (everyone), Private (subset), and Shared (external).
- **Policies:** Messaging, Meetings, and Apps.
