# SPFx & SharePoint AI Integration

As an SPFx expert, this is your most important section.

## 1. SPFx 1.21+ (The Agent Era)
The latest version of SPFx introduces **"Agentic Web Parts."**

*   **`BaseAgentWebPart`**: A specialized base class for building AI-aware UI components.
*   **Copilot UI Integration:** SPFx web parts can now surface in Copilot's response area.

## 2. Declarative Agents (No-Code)
You can build agents directly inside SharePoint lists and libraries.

*   **Customization:** Define the instructions, grounding data, and triggers.
*   **Publishing:** Agents can be shared with specific users or the whole organization.

## 3. Copilot Plugins (Pro-Code)
Extend M365 Copilot with your custom SPFx code.

*   **Message Extensions:** Allow Copilot to "search" and "act" on your custom data.
*   **Graph Connectors:** Bring your external data (SQL, CRM, etc.) into the M365 Semantic Index.
