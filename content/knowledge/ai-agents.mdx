# AI Agents Fundamentals

This is the "Brain" of your portal.

## 1. ReAct (Reasoning + Acting)
The ReAct pattern is the "gold standard" for agent architecture.

*   **Logic:** Think → Plan → Act → Observe → Repeat.
*   **The Power:** It allows an agent to solve multi-step problems by reasoning and then using a tool (e.g., search, SQL).

## 2. Tool Use (Function Calling)
Standardized way for agents to "click buttons" or "search data."

*   **Logic:** The LLM generates the arguments for a function, but the *code* executes it.
*   **The Goal:** Connect the LLM to real-world actions.

## 3. Memory Architectures
Agents need a way to remember user preferences across sessions.

*   **Short-term:** Buffer memory (recent chat).
*   **Long-term:** Semantic memory (stored in a Vector DB like Qdrant or Pinecone).
*   **Episodic:** Remembering specific past events.
