# Copilot Studio: Generative Actions

This is where the most aggressive AI updates are happening in 2026.

## 1. Generative Actions (GA)
Microsoft has moved beyond "Topics." Now, you can build **"Autonomous Agents."**

*   **Logic:** The agent uses LLM reasoning to decide which API or tool to call based on the user's intent.
*   **Benefits:** Reduces the need for building manual "Decision Trees" (topics).

## 2. Agent Orchestration
Build "Agent Teams" where a "Lead Agent" delegates tasks to "Specialists."

*   **SAP Agent:** Handles finance and invoicing.
*   **SharePoint Agent:** Handles document retrieval.
*   **HR Agent:** Handles employee leave and policy.

## 3. Dynamic Chaining
The ability for an agent to perform multi-step workflows autonomously.

*   **Example:** "Find the latest invoice, summarize the delta, and email the manager."
*   **The Power:** This happens without the human user prompting each step.
