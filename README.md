# AI Learning Portal (AI Agents + AB-900)

A lightweight, MDX-based self-learning platform built with **Next.js 15 App Router**, **Tailwind CSS**, and **Lucide icons**.

## Modules

- Dashboard (progress, daily task, 4-week plan, quick links)
- Knowledge Vault (AI Agents, Copilot Studio, Azure Foundry, Purview, SPFx)
- Exam Simulator (55 AB-900 practice questions with explanations)
- Resource Library (curated Microsoft Learn, GitHub, and YouTube)
- Study Buddy UI placeholder (ready for Azure AI Foundry / PydanticAI integration)

## Tech

- Next.js 15 (App Router)
- Tailwind CSS
- MDX + file system as content database

## Run

```bash
npm install
npm run dev
```

## Deploy

Ready for Vercel. For GitHub Pages, export/static adjustments can be added based on hosting mode.
